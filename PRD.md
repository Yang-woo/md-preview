# Markdown Preview - PRD

## 개요

마크다운 파일을 쉽고 빠르게 프리뷰할 수 있는 웹 애플리케이션

## 목표

- 로컬 마크다운 파일을 브라우저에서 즉시 프리뷰
- 실시간 편집 및 미리보기 지원
- GitHub 스타일 마크다운 + 확장 문법 지원

## 주요 기능

### 1. 파일 입력 방식

| 기능 | 설명 |
|------|------|
| 파일 선택 | 파일 탐색기에서 .md 파일 선택 |
| 드래그 앤 드롭 | 파일을 드롭존에 끌어다 놓기 |
| 직접 입력 | 텍스트 에디터에서 마크다운 직접 작성 |

### 2. 마크다운 렌더링

#### 기본 문법
- 헤딩 (h1-h6)
- 볼드, 이탤릭, 취소선
- 순서 있는/없는 리스트
- 링크, 이미지
- 인라인 코드, 코드 블록 (syntax highlighting)
- 인용문
- 수평선

#### GitHub Flavored Markdown (GFM)
- 테이블
- 체크박스 (task list)
- 자동 링크
- 취소선 (`~~text~~`)

#### 확장 기능
- **수식**: KaTeX 지원
  - 인라인: `$E = mc^2$`
  - 블록: `$$\sum_{i=1}^n i = \frac{n(n+1)}{2}$$`
- **다이어그램**: Mermaid 지원
  - 플로우차트
  - 시퀀스 다이어그램
  - 클래스 다이어그램
  - 간트 차트 등

### 3. UI/UX

#### 레이아웃
```
┌─────────────────────────────────────────────────────┐
│  [파일 선택] [새 문서]              Markdown Preview │
├────────────────────────┬────────────────────────────┤
│                        │                            │
│      에디터 영역        │       프리뷰 영역          │
│    (마크다운 입력)      │    (렌더링된 결과)         │
│                        │                            │
│                        │                            │
└────────────────────────┴────────────────────────────┘
```

#### 주요 UX 요소
- **실시간 프리뷰**: 입력과 동시에 렌더링 (debounce 적용)
- **스크롤 동기화**: 에디터와 프리뷰 스크롤 연동
- **다크/라이트 모드**: 테마 전환 지원
- **반응형**: 모바일에서는 탭 전환 방식

### 4. 추가 기능

| 기능 | 우선순위 | 설명 |
|------|----------|------|
| 파일 다운로드 | P1 | 편집한 마크다운을 .md로 저장 |
| HTML 내보내기 | P2 | 렌더링 결과를 HTML로 내보내기 |
| 전체화면 모드 | P2 | 에디터/프리뷰 각각 전체화면 |
| 단축키 지원 | P2 | Ctrl+B 볼드, Ctrl+I 이탤릭 등 |

## 기술 스택

| 구분 | 기술 |
|------|------|
| 프레임워크 | React + TypeScript |
| 빌드 도구 | Vite |
| 마크다운 파싱 | react-markdown |
| GFM 지원 | remark-gfm |
| 수식 렌더링 | remark-math + rehype-katex |
| 다이어그램 | mermaid |
| 코드 하이라이팅 | react-syntax-highlighter |
| 스타일링 | CSS Modules 또는 Tailwind CSS |

## 비기능 요구사항

- **성능**: 대용량 파일(10MB+)도 버벅임 없이 렌더링
- **접근성**: 키보드 네비게이션, 스크린리더 지원
- **브라우저 호환**: Chrome, Firefox, Safari, Edge 최신 2개 버전
