# PWA 아이콘

다음 아이콘 파일들이 필요합니다:

- `pwa-192x192.png` - 192x192 픽셀
- `pwa-512x512.png` - 512x512 픽셀
- `apple-touch-icon.png` - 180x180 픽셀
- `favicon.ico` - 32x32 픽셀

## 생성 방법

1. 온라인 도구 사용:
   - https://realfavicongenerator.net/
   - https://www.pwabuilder.com/imageGenerator

2. 또는 Photoshop/Figma에서 직접 생성

## 현재 상태

- ✅ `favicon.svg` - SVG 아이콘 (fallback)
- ⏳ PNG 아이콘들은 빌드 시 placeholder로 자동 생성됨

## 디자인 가이드

- 배경색: #2563eb (파란색)
- 심볼: "M" 또는 마크다운 아이콘
- 스타일: 미니멀, 모던
